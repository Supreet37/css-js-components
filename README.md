# css-js-components

A growing collection of small UI components and mini front-end projects — built with **plain HTML, CSS, and JavaScript** (no frameworks, no build step). Each folder is self-contained: just open its `index.html` in a browser.

Built as hands-on practice while preparing for frontend/full-stack placements — every component is small enough to build in one sitting but demonstrates a distinct real-world UI pattern.
