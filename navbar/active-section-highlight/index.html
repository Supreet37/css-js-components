Intro

Scroll down through each section — whichever one is in view gets highlighted in the nav automatically, using IntersectionObserver.

Features

No manual scroll-position math needed — the observer fires when a section crosses the viewport threshold.

Pricing

This pattern is standard on single-page marketing sites and docs pages with a sidebar TOC.

FAQ
